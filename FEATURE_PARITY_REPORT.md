# Canlifal Web -> Flutter Feature Parity Report

Tarih: 2026-06-09

## Kapsam ve kaynaklar

Bu rapor Canlifal web uygulamasındaki ozellikleri Flutter mobil uygulama ile
karsilastirir. Calisma alaninda gercek Next.js web istemci kaynak kodu yoktur;
bu nedenle analiz su kaynaklara dayanir:

- Uretim envanteri: `https://canlifal.com/canlifal-envanter-raporu.txt`
- Yerel API aynasi: `api/`
- Flutter uygulamasi: `mobile/`
- Deploy/parite dokumanlari: `docs/`
- Next.js uretim route referanslari: `docs/nextjs/`

Notlar:

- `api/` Express aynasi uretim web API'sinin tam kopyasi degildir.
- Uretim envanterinde 384 API endpoint ve 149 Prisma model oldugu belirtilir.
- Yerel `api/prisma/schema.prisma` daha kucuk bir semadir; bu nedenle DB
  tablolari alaninda uretim envanteri esas alinmis, yerel aynadaki farklar
  "Not" olarak isaretlenmistir.
- "Calisiyor mu?" kolonu web/uretim durumunu ve Flutter tarafindaki durumu
  birlikte yorumlar.

## Ozet durum

| Durum | Anlam |
|---|---|
| Var / calisiyor | Flutter tarafinda ilgili ekran, servis veya veri kaynagi mevcut ve yerel/CI tarafinda temel entegrasyon var. |
| Kismi | Flutter'da karsiligi var ancak uretim endpoint durumu, veri modeli veya davranis paritesi eksik/degisken. |
| Eksik | Flutter'da dogrudan karsiligi yok veya yalnizca link/web fallback var. |
| Web-only | Ozellik web/admin odakli; mobilde bilerek dogrudan karsilik yok. |

## En kritik eksikler

1. **Gercek web repo yok:** Web component dosya yolu ve tum 384 endpointin kod seviyesi dogrulamasi bu repoda yapilamiyor.
2. **Oyunlar:** Web'de cok genis; Flutter'da daha cok katalog/link seviyesi.
3. **Ajans, Bana Ozel, Reklam, Site Analitik, Admin panel:** Web/admin agirlikli; Flutter karsiligi sinirli veya yok.
4. **PK production durumu:** Flutter ve local mirror hazir; docs bazi production route'lari 404 olarak isaretliyor.
5. **Voice/live persistence:** Uretimde Prisma modelleri var; local mirror bazi kritik durumlari memory'de tutuyor.
6. **Membership/FCM/backgrounds endpoint durumu:** Dokumanlarda eski/yeni durumlar celisiyor; canli smoke test gerekli.
